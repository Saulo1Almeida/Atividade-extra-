# Atividade de Ponteiros em C

Este repositório contém a resolução das atividades sobre ponteiros em C.
Cada arquivo representa uma etapa da evolução proposta no enunciado.

## Arquivos

* `atv1.c`: usa um ponteiro para alterar diretamente o preço de um produto.
* `atv2.c`: usa uma função para aplicar desconto por passagem por referência.
* `atv3.c`: cria uma `struct Produto` e usa ponteiro para alterar o preço com o operador `->`.
* `atv4.c`: cria um array estático com 3 produtos e imprime os dados usando aritmética de ponteiros.
* `atv5.c`: usa `malloc` para cadastrar uma quantidade variável de produtos e depois libera a memória com `free`.

## Resumo

A atividade mostra a evolução do uso de ponteiros em C, começando com uma variável simples, passando por funções e structs, até chegar em arrays e alocação dinâmica de memória.
